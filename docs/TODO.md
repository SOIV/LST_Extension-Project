- Displaying
[ ] Show the transcript - When this is turned on, you can see the recognized speech as text.
[ ] Show the translated text - When this is turned on, you can see the translated text.
[ ] Hide the interim text - When this is turned on, the interim text is hidden
[ ] Prioritize listen in pop-up - When this is turned on, the recognized speech and the translation prioritize displaying in a pop-up window.

- Interface
[ ] Show Start/Stop Listening button in context menu - When this is turned on, you can start and stop listening using the context menu (right-click with your mouse). When you click "Save", the extension will automatically restart for the changes to take effect.
[ ] Hide translation tooltip - When this is turned on, you won't see on the top of options page a suggestion to translate the page
[ ] Hide Streaming Mode warning - When this is turned on, you won't see on the top of translation page that Streaming Mode is enabled.

- Console
[ ] Print the transcript to console - When this is turned on, you can open the Console by pressing Ctrl-Shift-J on Windows or Cmd-Option-J on a Mac to see the recognized speech as text.
[ ] Print the translated text to console - When this is turned on, you can open the Console by pressing Ctrl-Shift-J on Windows or Cmd-Option-J on a Mac to see the translated text.
